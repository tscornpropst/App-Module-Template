perl5-app-module-template
=========================

Yet Another Perl Module Starter

App::Module::Template

To install this module, run the following commands:

  perl Makefile.PL
  make
  make test
  make install

DEPENDENCIES
------------

This application requires Perl 5.16 or greater.

this application depends on Template Toolkit.
